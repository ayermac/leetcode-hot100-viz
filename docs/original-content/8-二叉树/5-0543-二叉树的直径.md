# 二叉树的直径：穿越树林的最长路径

## 生命的横跨：寻找树中的最大距离

想象你站在一片茂密的森林中，试图找出从一个极端到另一个极端最长能走多远。在二叉树的世界里，直径就是这样一种量度 —— 它不仅仅是一个数字，更是树的生命力和广度的见证。

## 直径的本质：不仅仅是长度

二叉树的直径（LeetCode第543题）是树中任意两个节点之间最长路径的长度。这个定义乍看可能简单，但蕴含着深刻的算法智慧。关键在于：
- 路径可以不经过根节点
- 长度定义为路径经过的边的数量
- 需要遍历整棵树才能找到最长路径

## 解题思路：寻径之旅的递归哲学

解决这个问题的核心是理解：树的直径 = 左子树深度 + 右子树深度

```go
var maxDiameter int

func diameterOfBinaryTree(root *TreeNode) int {
    maxDiameter = 0
    // 计算深度的同时更新直径
    calculateDepth(root)

    // 返回边的数量，而非节点数
    return maxDiameter
}

func calculateDepth(node *TreeNode) int {
    // 递归终止条件：空节点深度为0
    if node == nil {
        return 0
    }

    // 递归计算左右子树深度
    leftDepth := calculateDepth(node.Left)
    rightDepth := calculateDepth(node.Right)

    // 更新最大直径
    // 当前节点的直径 = 左子树深度 + 右子树深度
    if leftDepth+rightDepth > maxDiameter {
        maxDiameter = leftDepth + rightDepth
    }

    // 返回当前节点的最大深度
    if leftDepth > rightDepth {
        return leftDepth + 1
    }
    return rightDepth + 1
}
```

### 代码的深层逻辑解析

1. `maxDiameter` 全局变量
   - 记录树的最大直径
   - 在遍历过程中不断更新
   
2. `calculateDepth()` 方法
   - 同时完成两个任务：
     a. 计算节点深度
     b. 更新最大直径
   
3. 直径计算的关键：`leftDepth + rightDepth`
   - 代表穿过当前节点的最长路径
   - 不一定经过根节点

## 迭代方法：显式深度遍历

```go
func diameterOfBinaryTree(root *TreeNode) int {
    if root == nil {
        return 0
    }

    maxDiameter := 0
    // 存储节点的栈
    nodeStack := []*TreeNode{root}
    // 存储对应节点深度的栈
    depthStack := []int{0}

    for len(nodeStack) > 0 {
        node := nodeStack[len(nodeStack)-1]
        nodeStack = nodeStack[:len(nodeStack)-1]
        currentDepth := depthStack[len(depthStack)-1]
        depthStack = depthStack[:len(depthStack)-1]

        var leftDepth, rightDepth int
        if node.Left != nil {
            leftDepth = getNodeDepth(node.Left)
        }
        if node.Right != nil {
            rightDepth = getNodeDepth(node.Right)
        }

        if leftDepth+rightDepth > maxDiameter {
            maxDiameter = leftDepth + rightDepth
        }

        if node.Right != nil {
            nodeStack = append(nodeStack, node.Right)
            depthStack = append(depthStack, currentDepth+1)
        }

        if node.Left != nil {
            nodeStack = append(nodeStack, node.Left)
            depthStack = append(depthStack, currentDepth+1)
        }
    }

    return maxDiameter
}

// 辅助方法：获取节点深度
func getNodeDepth(node *TreeNode) int {
    if node == nil {
        return 0
    }
    leftDepth := getNodeDepth(node.Left)
    rightDepth := getNodeDepth(node.Right)
    if leftDepth > rightDepth {
        return 1 + leftDepth
    }
    return 1 + rightDepth
}
```

## 性能分析：算法的呼吸

### 时间复杂度：O(n)
- 每个节点仅访问常数次
- n为树中节点总数

### 空间复杂度：O(h)
- h为树的高度
- 最坏情况可达O(n)
- 最好情况（平衡树）为O(log n)
