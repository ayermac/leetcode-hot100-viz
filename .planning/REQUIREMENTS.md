# Requirements: LeetCode Hot 100 可视化学习平台

**Project Code:** LCVIZ
**Version:** MVP v1.0
**Last Updated:** 2026-04-30

---

## v1 Requirements (MVP)

### 基础结构 (STRUCT)

- [ ] **STRUCT-01**: 用户可以查看 17 个算法分类列表
  - 分类包括：哈希、双指针、滑动窗口、普通数组、矩阵、链表、二叉树、图论、回溯、二分查找、栈、堆、贪心算法、动态规划、多维动态规划、技巧
  - 每个分类显示题目数量、图标、简短描述

- [ ] **STRUCT-02**: 用户可以浏览每个分类下的题目列表
  - 显示题目编号、名称、难度（简单/中等/困难）
  - 支持按难度筛选

- [ ] **STRUCT-03**: 用户可以查看题目详情页
  - 题目描述（支持中文）
  - 难度标签、算法标签
  - LeetCode 原题链接

- [ ] **STRUCT-04**: 用户可以查看题目对应的题解代码
  - 默认显示 Go 语言实现
  - 语法高亮（Shiki）

- [ ] **STRUCT-05**: 用户可以切换代码语言
  - 支持：Go、Python、Java
  - 切换后保持当前滚动位置

### 核心可视化 (VIZ)

- [ ] **VIZ-01**: 用户可以查看数组的可视化展示
  - 元素显示为方块，带索引
  - 支持状态颜色：正常、比较中、交换中、已排序、pivot
  - 支持指针箭头标注

- [ ] **VIZ-02**: 用户可以查看链表的可视化展示
  - 节点显示为方块，带值
  - 箭头连接 next 指针
  - 支持慢快指针、循环检测展示

- [ ] **VIZ-03**: 用户可以控制动画播放
  - 播放/暂停按钮
  - 单步前进/后退
  - 重置按钮
  - 速度控制（0.25x, 0.5x, 1x, 2x）

- [ ] **VIZ-04**: 用户可以修改输入数据
  - 文本输入框
  - 预设测试用例选择
  - 输入验证和错误提示

- [ ] **VIZ-05**: 代码与动画同步
  - 当前执行行高亮
  - 步骤描述文字

### 用户体验 (UX)

- [ ] **UX-01**: 用户可以搜索题目
  - 支持按题目名称搜索
  - 支持模糊匹配
  - 实时搜索结果

- [ ] **UX-02**: 用户可以筛选题目
  - 按难度筛选（简单/中等/困难）
  - 按分类筛选
  - 按完成状态筛选

- [ ] **UX-03**: 用户可以收藏题目
  - 收藏/取消收藏按钮
  - 收藏列表页面
  - 数据存储在 localStorage

- [ ] **UX-04**: 用户可以切换暗色/亮色模式
  - 模式切换按钮
  - 记住用户偏好
  - 所有可视化适配两种模式

- [ ] **UX-05**: 网站响应式设计
  - 支持桌面端（主要）
  - 支持平板端
  - 移动端基础支持（简化交互）

| 移动端原生 App | Web 优先 |
